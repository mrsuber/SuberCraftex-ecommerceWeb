# Comprehensive Tailoring Education Curriculum

## Program Overview

This curriculum takes apprentices from complete beginner to professional tailor through a structured 5-level program. Each level builds upon the previous, ensuring mastery of fundamental skills before advancing to more complex techniques.

**Program Duration:** 18-24 months (self-paced)
**Certification Levels:** Apprentice, Journeyman, Craftsman, Master Tailor, Artisan Master

#### Fabric Characteristics

**Weave Types**
1. **Plain Weave** - Simple over-under pattern, most common
2. **Twill Weave** - Diagonal pattern, durable (denim, gabardine)
3. **Satin Weave** - Smooth, lustrous surface
4. **Herringbone** - V-shaped twill pattern
5. **Houndstooth** - Broken check pattern
6. **Birdseye** - Small diamond pattern
7. **Pinstripe** - Vertical fine lines
8. **Windowpane** - Large check pattern
9. **Glen Plaid** - Complex check pattern
10. **Prince of Wales** - Large check with overcheck

#### Grain Lines and Pattern Matching

**Grain Line Rules**
- **Lengthwise Grain (Warp):** Runs parallel to selvage, strongest direction
- **Crosswise Grain (Weft):** Runs perpendicular to selvage
- **Bias:** 45-degree angle, maximum stretch
- **True Bias:** Exact 45-degree angle

**Pattern Matching Requirements**
| Pattern Type | Matching Points | Difficulty |
|--------------|-----------------|------------|
| Solid | None | Easy |
| Pinstripe | Vertical alignment | Moderate |
| Horizontal Stripe | Side seams, center back | Moderate |
| Plaid/Check | All seams, pockets | Difficult |
| Herringbone | Chevron at seams | Difficult |
| Large Prints | Visual balance | Expert |

### Module 1.3: Hand Stitching Fundamentals

**Duration:** 4 weeks

#### Essential Hand Stitches

**1. Running Stitch**
```
Purpose: Temporary basting, ease stitching
Stitch Length: 1/4" - 1/2"
Technique: In-out motion, even spacing
Applications: Gathering, temporary seams
```

**2. Basting Stitch**
```
Purpose: Temporary hold for fitting
Stitch Length: 1/2" - 1"
Technique: Long running stitch
Removal: After permanent stitching
Thread: Contrasting color for visibility
```

**3. Backstitch**
```
Purpose: Strong permanent seam
Stitch Length: 1/8"
Technique: Forward, back, forward
Applications: Seam repair, stress points
```

**4. Slip Stitch (Ladder Stitch)**
```
Purpose: Invisible closures
Stitch Length: 1/4" - 3/8"
Technique: Catch fold edge, then base
Applications: Lining attachment, hems
Visibility: Nearly invisible when done correctly
```

**5. Catch Stitch (Herringbone Stitch)**
```
Purpose: Flexible hem securing
Stitch Length: 1/2" - 3/4"
Technique: Alternating diagonal stitches
Applications: Hems, interfacing, stretch seams
Characteristics: Allows fabric movement
```

**6. Blind Hem Stitch**
```
Purpose: Invisible hemming
Stitch Length: 1/4"
Technique: Catch single thread of outer fabric
Applications: Trouser hems, jacket hems
Visibility: Invisible from right side
```

**7. Fell Stitch**
```
Purpose: Attaching linings
Stitch Length: 1/8" - 1/4"
Technique: Angled stitches into fold
Applications: Lining edges, facings
Strength: Moderate to strong
```

**8. Pad Stitch**
```
Purpose: Shaping lapels and collars
Stitch Length: 1/4" - 3/8"
Technique: Diagonal rows while rolling fabric
Applications: Lapel roll, collar stands
Skill Level: Advanced
```

**9. Pick Stitch**
```
Purpose: Decorative edge finishing
Stitch Length: Tiny (1/16")
Technique: Surface stitch, minimal thread show
Applications: Lapel edges, pocket edges
Visibility: Small dots visible
```

**10. Buttonhole Stitch**
```
Purpose: Buttonhole edges
Technique: Closely spaced blanket stitch
Density: 16-24 stitches per inch
Applications: Hand buttonholes
Skill Level: Intermediate to Advanced
```

**11. Bar Tack**
```
Purpose: Reinforcement at stress points
Technique: Parallel threads with wrapping
Applications: Pocket corners, vent bases
Strength: Very strong
```

**12. Cross Stitch (Tailor's Tack)**
```
Purpose: Pattern marking
Technique: Loose X stitches through layers
Applications: Marking darts, notches
Removal: After cutting apart
```

**Jacket/Coat Measurements**

|-------------|--------|------------|

| Sleeve Length | Shoulder point to wrist | Arm slightly bent |

### Module 1.5: Pattern Basics

**Duration:** 3 weeks

#### Pattern Symbols and Markings

| Symbol | Meaning |
|--------|---------|
| Solid line | Cutting line |
| Dashed line | Stitching line |
| Double-headed arrow | Grain line |
| Triangle notch | Matching point |
| Circle | Dot marking (darts, etc.) |
| Curved arrow | Roll line |
| Square | Button placement |
| Buttonhole line | Buttonhole position |

#### Pattern Pieces - Basic Jacket

1. **Front** - Main body, includes dart placements
2. **Back** - Center back, may be one or two pieces
3. **Side Panel** - If applicable, creates shape
4. **Upper Sleeve** - Two-piece sleeve top
5. **Under Sleeve** - Two-piece sleeve bottom
6. **Upper Collar** - Top collar piece
7. **Under Collar** - Collar stand piece
8. **Front Facing** - Inside front finish
9. **Welt Pocket** - Pocket pieces
10. **Chest Piece** - Canvas interlining

#### Pattern Alterations - Foundation Level

**Simple Adjustments**
1. Lengthening/Shortening
   - Add/remove at adjustment lines
   - Maintain grain alignment
   - True seam lines after

2. Width Adjustments
   - Add/remove at side seams
   - Distribute evenly
   - Adjust corresponding pieces

### Module 2.3: Pressing Techniques

**Duration:** 2 weeks

#### Pressing vs. Ironing

| Pressing | Ironing |
|----------|---------|
| Lift and lower iron | Slide iron back and forth |
| Used during construction | Used for finishing |
| Sets seams and shapes | Removes wrinkles |
| Uses press cloth | May be direct |

#### Pressing Equipment

1. **Iron**: Steam capability, temperature control
2. **Pressing Board**: Covered, padded surface
3. **Sleeve Board**: Narrow, for sleeves
4. **Tailor's Ham**: Curved surface for darts, curves
5. **Seam Roll**: Cylindrical, for seams
6. **Point Presser**: For collar points, corners
7. **Clapper**: Wooden block for setting creases
8. **Velvet Board/Needle Board**: For pile fabrics
9. **Press Cloth**: Protects fabric from shine

#### Pressing by Fabric Type

| Fabric | Temperature | Steam | Press Cloth | Notes |
|--------|-------------|-------|-------------|-------|
| Cotton | High | Yes | Optional | Can press directly |
| Linen | High | Yes | Optional | Press while damp |
| Wool | Medium-High | Yes | Always | Never slide iron |
| Silk | Low | Light/None | Always | Test first |
| Synthetic | Low | Light/None | Recommended | Heat damage risk |

#### Pressing Operations

**Seam Pressing Sequence**
1. Press seam flat as sewn (sets stitches)
2. Press seam open
3. Press from right side with cloth

**Dart Pressing**
1. Press flat as sewn
2. Press toward center (vertical) or down (horizontal)
3. Shape over ham for curves

**Edge Pressing**
1. Understitch if applicable
2. Turn and press edge
3. Use point presser for corners

4. **Attach to Front**

   - Baste in place
   - Seam allowances free for later manipulation

### Module 3.3: Sleeve Setting

**Duration:** 4 weeks

#### Sleeve Theory

**Sleeve Pitch**: The angle at which the sleeve hangs
- Forward pitch: Comfortable for most people
- Neutral pitch: Arms hang straight down
- Back pitch: Unusual, may indicate pattern issue

**Ease Distribution**:
- Total ease: 1" - 1.5" for set-in sleeve
- Distribution: Majority at cap, less at notches
- Ease stitches: Two rows, basted

#### Two-Piece Sleeve Construction

1. **Prepare Sleeve**
   - Sew front seam (under sleeve to upper sleeve)
   - Press seam open
   - Sew back seam
   - Press seam open
   - Create vent

2. **Sleeve Head**
   - Cut domette/wadding for sleeve head
   - Baste sleeve head roll
   - Shape creates smooth cap

3. **Setting the Sleeve**
   - Match notches and ease marks
   - Pin at key points (underarm, cap, notches)
   - Baste for fitting
   - Hand-stitch or machine for final

#### Fitting the Sleeve

**Issues and Corrections**:

| Problem | Cause | Solution |
|---------|-------|----------|
| Pulls at cap | Sleeve too narrow | Increase sleeve width |
| Diagonal wrinkles | Wrong pitch | Adjust rotation |
| Horizontal wrinkles | Too much ease at cap | Reduce cap height |
| Binding at underarm | Armhole too high | Lower armhole |

### Module 3.4: Advanced Pocket Construction

**Duration:** 3 weeks

#### Double Welt (Jetted) Pocket

**Standard Dimensions**:
- Opening: 5.5" - 6"
- Welt height: 1/4" - 3/8" each
- Total opening: 1/2" - 3/4"

**Construction Steps**:

1. **Mark Position**
   - Draw pocket line on right side
   - Mark ends precisely

2. **Prepare Welts**
   - Cut welt strips (1" wide plus seam allowances)
   - Interface if needed
   - Fold and press

3. **Attach Welts**
   - Position welts with raw edges at center line
   - Stitch exactly on marked lines
   - Backstitch at ends

4. **Cut Opening**
   - Cut on center line, stopping 1/2" from ends
   - Cut diagonally to corners (triangle tabs)

5. **Turn and Press**
   - Turn welts through opening
   - Press carefully
   - Secure triangle tabs

6. **Pocket Bag**
   - Attach pocket bag pieces to welts
   - Stitch bag closed
   - Secure to jacket

#### Flap Pocket

**Adds to jetted pocket**:
1. Construct flap (two pieces plus interfacing)
2. Insert flap before cutting opening
3. Flap covers welts when finished

### Module 3.5: Fitting Adjustments

**Duration:** 4 weeks

#### Common Fitting Issues

**Shoulder Issues**

| Problem | Symptom | Solution |
|---------|---------|----------|
| Square shoulders | Wrinkles at neck | Raise shoulder seam |
| Sloping shoulders | Puckers at shoulder | Lower shoulder seam |
| Narrow shoulders | Pulls across back | Let out shoulder seams |
| Broad shoulders | Sleeve falls off shoulder | Take in shoulder seams |
| Uneven shoulders | One side pulls | Adjust uneven side only |

**Back Issues**

| Problem | Symptom | Solution |
|---------|---------|----------|
| Sway back | Excess fabric at waist | Reduce back length at waist |
| Round back | Pulls at upper back | Add length at upper back |
| Erect back | Excess at upper back | Reduce back length |
| Prominent blades | Pulling at blades | Add width, shape darts |

**Front Issues**

| Problem | Symptom | Solution |
|---------|---------|----------|
| Prominent chest | Pulls across chest | Add front length/width |
| Flat chest | Excess fabric | Reduce front length/width |
| Large bust | Diagonal wrinkles | Increase bust dart |
| Prominent stomach | Pulls at waist | Add front length |

#### Adjustment Techniques

**Pattern Adjustment vs. Fitting Adjustment**
- Pattern adjustment: Before cutting
- Fitting adjustment: During construction
- Both may be needed

**The Fitting Sequence**
1. **First Fitting (Basted)**
   - Check overall length
   - Check shoulder position
   - Check ease and balance
   - Mark collar position

2. **Second Fitting (Partial Assembly)**
   - Check sleeve position
   - Verify pocket placement
   - Confirm button stance
   - Check lapel roll

3. **Final Fitting (Nearly Complete)**
   - Verify hem length
   - Check lining fit
   - Final button position
   - Overall appearance

### Module 3.6: Finishing Techniques

**Duration:** 3 weeks

#### Hand Finishing

**Pick Stitching**
- Used on lapels, pockets, front edge
- Tiny stitches visible on right side
- Spacing: 1/4" - 3/8" apart
- Use silk buttonhole twist

**Buttonholes - Hand Worked**
1. Cut opening
2. Work buttonhole stitch around opening
3. Create bar tacks at ends
4. Use buttonhole gimp for body
5. Stitches per inch: 16-24

**Button Attachment**
1. Position button
2. Shank: 1/8" - 1/4" depending on fabric weight
3. Wrap shank
4. Anchor on wrong side

#### Lining Installation

**Lining Construction**
1. Construct lining body
2. Leave opening at bottom of one seam
3. Install lining through opening
4. Slip stitch closed
5. Tack at key points (vents, armholes)

**Free-Hanging Lining**
- Bottom edge not attached
- Allows fabric and lining to move independently
- Jump hem: Lining 1/2" shorter than jacket

#### Final Pressing

**Sequence**
1. Press all interior seams
2. Press exterior seams
3. Shape collar and lapels
4. Press sleeves
5. Set creases where appropriate
6. Steam entire garment
7. Allow to dry on hanger

## Certification Requirements

### Apprentice Certificate (Level 1)
- Complete all Level 1 assignments
- Pass Level 1 Final Project with 75%+
- Time: 3 months minimum

### Journeyman Certificate (Level 2)
- Hold Apprentice Certificate
- Complete all Level 2 assignments
- Pass Level 2 Final Project with 80%+
- Time: 4 months additional minimum

### Craftsman Certificate (Level 3)
- Hold Journeyman Certificate
- Complete all Level 3 assignments
- Pass Level 3 Final Project with 85%+
- Time: 5 months additional minimum

### Master Tailor Certificate (Level 4)
- Hold Craftsman Certificate
- Complete all Level 4 assignments
- Pass Level 4 Final Project with 90%+
- Time: 6 months additional minimum

### Artisan Master Certificate (Level 5)
- Hold Master Tailor Certificate
- Complete all Level 5 assignments
- Pass Master Portfolio with 90%+
- Time: 6 months additional minimum
- Demonstrated teaching ability
- Original design contribution
